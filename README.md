# Next.js React 19 – Optimized (React Compiler)

## Beschreibung
Dieses Repository enthält die **fortschrittlichste Variante** der Landingpage mit **Next.js und React 19**, inklusive React Compiler.

## Zweck im Rahmen der Bachelorarbeit
Diese Version untersucht das Zusammenspiel von:
- Next.js
- React Compiler
- automatisierten Performance-Optimierungen

## Implementierte Optimierungen
- Alle Next.js Optimierungen
- React Compiler
- Reduzierte manuelle Memoization
- Optimierte Bildauslieferung
- Tailwind CSS

## Technischer Stack
- React 19
- Next.js
- Tailwind CSS
- React Compiler
